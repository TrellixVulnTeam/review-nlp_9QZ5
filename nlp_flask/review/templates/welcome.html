How it works?

This predictor is built by Python using the LSTM neural network. You can you Jupyter Notebook to creat the mode and evaluate the accuracy. For this project, the best accuracy I got is 87.36%, if you want to reach higher accuracy, you may need more computing resource. After generating the model, it is a good choice to use Flask to deploy the project on a server as a web application.
